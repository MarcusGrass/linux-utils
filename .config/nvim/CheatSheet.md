# Usage Cheat sheet

##  Nvimtree

- E: Expand all
- W: Collapse all
- s: open with system (will likely open directories with firefox, f.e., good for PDFs)
- "-" dir up
- f: find
- F: close find

- a: Create, end with slash for dir
- r: Rename file
- d: Delete file or dir
- x: Cut fir or dir
- c: Copy
- p: Paste

- y: copy file name
- Y: copy relative path
- g + y: copy abs path
